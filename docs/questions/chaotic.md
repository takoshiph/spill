# Chaotic — Question Bank

Fast, silly, unpredictable — but still pulling real things out of people. More wildcards than other vibes. Grouped by **depth tier** (deck deals Warm-up → Open up → Deep; even chaos warms up before it bites). Theme tags in `[brackets]` are for balancing only. Edit freely.
